# Todo App (React + Vite)

Простий застосунок списку задач, створений з використанням React та Vite.

## Функціонал

- Відображення списку задач
- Список задач заданий як константа
- Використання компонентів (TodoList, TodoItem)
- Передача даних через Props
- Рендер списку через map
- Базове стилізування

## Структура

src/
- App.jsx
- main.jsx
- index.css
- components/
  - TodoList.jsx
  - TodoItem.jsx

## Запуск проєкту

1. Встановити залежності:

npm install

2. Запустити сервер:

npm run dev

3. Відкрити в браузері:

http://localhost:5173/
